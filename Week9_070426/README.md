# Week8 AngularJS Hello World Project

A simple HTML project demonstrating AngularJS basics including ng-controller, ng-model, and expressions.

## Features
- Input field bound with ng-model
- Dynamic greeting using expressions
- Conditional display with ng-if
- Simple calculator with numeric inputs and expressions

## How to Run
Open `index.html` in a web browser. No server required as it uses AngularJS from CDN.

## Technologies
- AngularJS 1.8.2
- HTML5
- CSS
